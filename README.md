# Java Fullstack Learning - Payment System
  ## Session 1: OOP - Abstract classes, Inheritance, Polymorphism
  - `Payment` (abstract base class)
  - `CreditCardPayment`
  - `WalletPayment`
  - `UPIPayment`
  - `Main` (driver class)
    ### Run Locally
    From the project root:
    ```bash
    javac -d out src/com/paymentz/*.java
    java -cp out com.paymentz.Main
    ```
    ### Concepts Practiced
    - Inheritance
    - Method overriding
    - Polymorphism
    - Collections (`ArrayList`)
